Le service Asteroids - NeoWs (Near Earth Object Web Service) de la NASA est une API RESTful fournissant des informations sur les astéroïdes proches de la Terre. Cette API est disponible depuis le 6 juin 2016, offrant ainsi un accès aux données sur les objets proches de la Terre collectées par l'équipe des astéroïdes du Jet Propulsion Laboratory (JPL) de la NASA.
